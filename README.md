# battleship
## Prerequisites ##
* Install Node.js version 18.16.0
* Clone this repository: https://github.com/nastyavolkm/battleship.git
* Switch to develop branch
  *Use npm install
## Program start ##
* DEV mode:

    * npm run start:dev

* PROD mode:
    * npm run start:prod

### Open 2 tabs in your browser on localhost:8181 to check the task ###
